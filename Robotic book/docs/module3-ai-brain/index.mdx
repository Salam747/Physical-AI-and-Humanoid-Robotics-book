# Module 3 – The AI-Robot Brain (NVIDIA Isaac™)

This module delves into building the "AI-Robot Brain" using NVIDIA Isaac Sim and Isaac ROS. You will master GPU-accelerated perception, navigation, and manipulation techniques, including domain randomization, Isaac ROS GEMs (Visual SLAM, AprilTag, PeopleNet), reinforcement learning for robotics, and effective sim-to-real transfer strategies.
